# My Wee Wander Planner

**My Wee Wander Planner** is a customisable, all-in-one web app for planning adventures: organise itineraries, packing, budgets, notes, documents and travel photos.

##  Features

- **Itinerary Builder** — Create and organise trip events by date and time.
- **Packing List** — Track essential items and travel tips.
- **Budget Tracker** — Categorise and manage travel expenses.
- **Notes Section** — Jot down reminders, plans, and travel thoughts.
- **Documents Manager** — Securely store travel documents and emergency contacts.
- **Photo Gallery** — Upload and save memories from your trip.

##  Take a look!

https://my-travel-planner-virid.vercel.app/

##  Usage Overview

- **Dashboard:** View and access different planner sections.
- **Itinerary:** Add/manage trip events (flights, activities, etc.).
- **Packing:** List essential items & plan travel readiness.
- **Budget:** Track and classify travel expenses.
- **Notes:** Save important info (meal plans, reminders).
- **Documents:** Store critical travel documents & contacts.
- **Photos:** Safely upload & organize your travel snaps.

##  Tech Stack

- **Frontend:** React (Next.js)
- **Styling:** CSS / Styled Components
- **Hosting:** Vercel / Webapp platform
- **Back End:** Firebase (for auth and data storage)
